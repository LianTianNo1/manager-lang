# manager-lang


> 学习河畔老师的课程，
>
> 使用Vue3+ElementPlus+Koa2+mongoDB开发后台管理系统
>
> 服务端项目地址：[后台](https://github.com/LianTianNo1/manager-lang-server)
>
> 菜单动态渲染，登录权限验证，申请和审批工作流，角色管理，部门管理...

### 安装

克隆

```bash
git clone --depth=1 https://hub.fastgit.org/LianTianNo1/manager-lang.git
```

安装依赖

```bash
cd manager-lang
yarn add
```

运行

```bash
npm run dev
```
